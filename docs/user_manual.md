# User Manual

## Getting Started

1. Open the web application in your browser
2. Click "Choose File" to select an image of a fruit
3. Click "Grade Fruit" to process the image
4. View the grade and confidence score

## Supported Fruits

- Mangoes
- Apples
- Oranges
- Tomatoes

## Grade Definitions

- **Grade A**: Excellent quality, no defects
- **Grade B**: Good quality, minor imperfections
- **Grade C**: Poor quality, significant defects
